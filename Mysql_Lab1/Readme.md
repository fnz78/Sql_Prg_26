# 📚 MySQL Library Management System (Lab Record)

## 📌 Overview
This project is a MySQL-based Library Management System created as part of a college lab record.

It demonstrates:
- Database creation
- Table relationships using foreign keys
- Data insertion
- Constraints (PRIMARY KEY, FOREIGN KEY, CHECK)
- ALTER operations

---

## 🏗️ Database Structure

### Tables:
- Students
- Membership
- Booktype
- Book
- Bookissue

---

## 🔗 Relationships

- One Student → One Membership
- One Booktype → Many Books
- One Member → Many Book Issues
- One Book → Many Issues
